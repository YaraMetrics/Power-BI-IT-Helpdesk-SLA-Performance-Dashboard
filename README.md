Dieses Projekt wurde entwickelt, um zentrale Herausforderungen im IT‑Support zu lösen: fehlende Transparenz, lange Bearbeitungszeiten und unklare SLA‑Einhaltung.
Das Dashboard analysiert über 900 Tickets und bietet:
• vollständige Übersicht über Ticket‑Volumen, Typen und Prioritäten
• Vergleich der Performance pro Team, Department und Agent
• Analyse der durchschnittlichen Lösungszeit (39,7 h)
• klare Darstellung von SLA‑Verstößen und Ursachen
• Filter nach Zeitraum, Service, Agent und Ticketart
Durch diese Transparenz können Unternehmen Engpässe schneller erkennen, Ressourcen gezielt einsetzen und datenbasierte Entscheidungen treffen.
Tech Stack:
Power BI · DAX · Data Modeling · ETL · Prozessanalyse
